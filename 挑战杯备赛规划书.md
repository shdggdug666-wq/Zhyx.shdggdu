## 1. 赛事认知:先搞清楚在打什么仗

### 1.1 大挑 vs 小挑(一图看懂)

| 维度 | 大挑(课外学术科技作品竞赛) | **小挑(创业计划竞赛)← 2026 是这个** |
|---|---|---|
| 举办年 | 奇数年 | **偶数年(2026)** |
| 核心作品 | 论文 / 调研报告 / 科技发明说明书 | **商业计划书(BP)+ 创业项目** |
| 评审重心 | 学术性、科学性、创新性 | **商业模式可行性、团队、市场、社会价值** |
| 呈现形式 | 论文 + 答辩 | BP + 路演(PPT)+ 答辩 |
| 含金量 | 保研/评奖重要加分项 | 同等量级,创业类赛事中最高规格 |

> 一句话:小挑比的不是「你研究出了什么」,而是「**你这个生意能不能做成、凭什么由你做成**」。

### 1.2 2026 小挑的五大组别(选题必须落到其中一个)

围绕「**创新、协调、绿色、开放、共享**」五大发展理念,通常设五个赛道:

| 组别 | 适合的项目类型 | 典型例子 |
|---|---|---|
| **科技创新和未来产业** | 硬科技、新材料、人工智能、生物医药、未来产业 | AI 大模型应用、新能源器件、合成生物 |
| **乡村振兴和产业发展** | 农业科技、县域经济、乡村文旅、助农 | 智慧农业、特色农产品品牌、非遗活化 |
| **城市治理和社会服务** | 智慧城市、适老化、医疗健康、教育、社区服务 | 智慧社区平台、银发经济、慢病管理 |
| **生态环保和可持续发展** | 双碳、环保技术、循环经济、清洁能源 | 碳核算工具、固废资源化、绿色供应链 |
| **文化创意和区域合作** | 文化 IP、数字文创、区域协同、跨境 | 国潮品牌、数字文博、区域特产整合 |

> 组别选择直接决定评委池和评审视角,§4 会讲怎么选。

### 1.3 评审逻辑(通用评分维度,各年细则微调)

历年小挑评审通常围绕以下几个维度,权重随届次调整:

1. **创新性**(技术/模式/产品是否有真创新,不是「换个包装」)
2. **市场与商业模式**(痛点是否真实、市场多大、怎么赚钱、能否规模化)
3. **可行性**(资源、技术、财务、落地路径是否站得住)
4. **团队**(专业互补、执行能力、与项目的匹配度)
5. **社会价值**(契合五大发展理念、就业带动、公益属性)

> 评委大多是投资人、创业导师、行业专家——**用「这能不能投、能不能做」的视角审稿**,而非学术论文视角。

### 1.4 赛道层级与节奏

`校赛 → 省赛(省级初赛/复赛/决赛)→ 国赛(全国初赛/终审决赛)`

- **校赛**:学校自己组织,淘汰率较高,是「能不能出校门」的关键门槛。
- **省赛**:各省组织,金奖/银奖/铜奖;省赛金奖才有较大概率推国赛。
- **国赛**:下半年举行,金奖/银奖/铜奖,另有「挑战杯」流动杯(集体荣誉)。

### 2.3 节奏判断(针对「6 月才起步」的情况)

6 月起步属于**偏晚但完全来得及**(很多金奖项目也是暑假才真正发力),但意味着:
- **暑假是黄金期**——必须高强度推进项目验证和 BP;
- 不能等「想完美了再动」——选题边做边收敛;
- 指导老师、场地、经费等资源**放假前就要谈好**,别等开学。

## 3. 阶段任务分解(主干)

### 阶段 0 ｜ 诊断与立项(现在 ~ 7 月初)

- [ ] 通读本文档,建立赛事认知(§1)
- [ ] 查清本校时间节点(§2.2)
- [ ] 盘点自身资源:专业、技能、导师人脉、已有成果(大创项目、课程设计、实验室成果等)
- [ ] 列出 3-5 个候选方向,用 §4 自检清单筛
- [ ] **产出物**:一页纸「项目立项说明」(谁、做什么、哪个组别、初步判断为什么能成)

### 阶段 1 ｜ 团队组建(7 月上旬)

- [ ] 核心团队 **5-8 人**(太多难协调,太少能力不够);角色互补:
  - 项目负责人(CEO,统筹+答辩)
  - 技术/产品(把东西做出来)
  - 市场/运营(调研、商业模式)
  - 财务(财务模型、成本测算)
  - 设计/文案(PPT、BP 美化、视觉)
- [ ] 找 **1-2 位指导老师**:至少一位专业对口的老师负责学术/技术把关,最好还有一位有产业/创业经验的老师
- [ ] 建立协作机制:飞书/钉钉/腾讯文档 + 每周固定例会
- [ ] **产出物**:团队花名册(姓名/学院/负责模块/联系方式)、指导老师确认

> 💡 团队互补是小挑评分点。一个清一色同专业同年级的队伍会被扣分;「技术 + 商科 + 设计」的跨界组合更受评委欢迎。

### 阶段 2 ｜ 选题与组别匹配(7 月中下旬)

- 详见 **§4**。核心动作:
- [ ] 在阶段 0 候选方向基础上,做 10-15 人次目标用户访谈,验证痛点真实性
- [ ] 对照五大组别,锁定**一个**主赛道
- [ ] 查重:在知网/专利库/往届获奖项目里查,确认没有高度雷同
- [ ] **产出物**:最终选题说明 + 组别确认(一句话能说清,且别人听不出明显漏洞)

### 阶段 3 ｜ 项目验证(7 月底 ~ 8 月)

这一步是把「PPT 创业」变成「有据可查的项目」,是小挑区别于水赛的关键。

- [ ] **用户调研**:问卷 + 深访,拿到可引用的数据(「87% 的受访 X 表示……」)
- [ ] **最小可行产品(MVP)/原型**:能演示的最小形态——可以是 demo、3D 模型、流程图、小程序雏形
- [ ] **初步财务模型**:成本结构、收入来源、定价、盈亏平衡测算(哪怕粗糙,要有逻辑)
- [ ] **落地证据**:合作意向书、试点数据、用户反馈、已有营收/订单(有则强加分,没有不强求)
- [ ] **产出物**:调研报告、原型 demo、财务测算表

### 阶段 4 ｜ 商业计划书(BP)撰写(8 月 ~ 9 月初)

- 详见 **§5**。BP 是小挑的**核心交付物**,通常 20-40 页。
- [ ] 按 §5 模板逐模块填,先骨架后血肉
- [ ] 内部交叉评审:团队成员互相挑刺
- [ ] 指导老师过审
- [ ] **产出物**:BP v1(完整、逻辑闭环、无错别字/错数)

### 阶段 5 ｜ 材料与路演打磨(9 月)

- [ ] 路演 PPT(10-15 页,与 BP 同源但更精炼、视觉化)
- [ ] 答辩稿(5 分钟路演 + 3-5 分钟答辩 Q&A)
- [ ] **至少 3 轮模拟答辩**:请老师、学长、甚至非本专业同学当评委提刁钻问题
- [ ] 一句话电梯演讲(elevator pitch)练到脱口而出
- [ ] **产出物**:终版 PPT、答辩稿、模拟答辩问题清单及标准回答

### 阶段 6 ｜ 各赛段迭代(校赛后)

- 校赛 → 省赛 → 国赛,**每升一级都要按评委反馈大改一轮**,不是「同一份材料打到底」。
- [ ] 校赛后:收集评委意见,改 BP 和路演
- [ ] 省赛后:同上,且通常要补强「社会价值」「落地证据」
- [ ] 国赛:视觉、数据、故事全面升级,准备应对最严苛的提问

## 4. 选题方法论(从零最关键的一步)

> 小挑圈流传一句话:「**选题定生死**」。选题错了,BP 写得再漂亮也难翻盘。

### 4.1 好选题的五个来源

1. **自身专业/实验室成果转化**:最靠谱——有技术壁垒、有导师背书、有数据(大挑/大创成果直接转小挑很常见)。
2. **真实痛点观察**:自己或身边人遇到的麻烦(校园、社区、实习单位、家乡)。
3. **政策风口**:翻近 1-2 年的国家/地方政策文件(双碳、乡村振兴、银发经济、低空经济、AI+、新质生产力),政策指哪打哪。
4. **往届获奖项目的「相邻空间」**:研究近 3 届国赛金奖项目,找它们的未覆盖人群/场景/技术升级方向(不是抄,是找空隙)。
5. **产业资源**:导师/家人/校友在某个行业,能拿到真实场景和数据。

### 4.2 选题自检清单(过不了就别做)

- [ ] **真实痛点**:有没有真实的人愿意为这个付钱/付出?(不是「我觉得有需求」)
- [ ] **市场够大**:目标市场规模撑得起一个生意?(有可引用的数据)
- [ ] **真创新**:技术、模式、产品里至少有一项,别人没做过或没你做得好?(不是「换个皮」)
- [ ] **可落地**:在现有技术、资源、法规下能做出来,不是科幻?
- [ ] **团队匹配**:你们团队**凭什么**做这个?(专业、资源、关系网)
- [ ] **社会价值**:能挂上五大发展理念中的哪一条?(挂不上很难拿高分)
- [ ] **不撞车**:知网/专利/往届项目检索,没有高度雷同的获奖项目?

### 4.3 五大组别怎么选

| 你的项目强在… | 建议组别 |
|---|---|
| 硬核技术/前沿科技 | 科技创新和未来产业 |
| 涉农/县域/家乡 | 乡村振兴和产业发展 |
| 民生/公共服务/健康 | 城市治理和社会服务 |
| 环保/双碳/新能源 | 生态环保和可持续发展 |
| 文化 IP/区域品牌/跨境 | 文化创意和区域合作 |

> 经验:**不要哪个组别看起来「高级」就选哪个**,而要选**项目最契合、竞争对手相对少、评委最能 get**的。一个智慧农业项目放进「乡村振兴」往往比放进「科技创新」更容易出彩。

### 4.4 选题避坑

- ❌ 「大而全」:想解决所有人的所有问题 → 砍到一个具体场景、一个具体人群。
- ❌ 「纯想法,无验证」:只有点子没有数据/原型 → 评委最怕。
- ❌ 「技术自嗨」:技术很强但说不清卖给谁、怎么赚钱。
- ❌ 「伪需求」:自己想象出来的需求,市场不买单。
- ❌ 「红海硬刚」:成熟赛道没有差异化壁垒,拼不过既有玩家。

## 5. 商业计划书(BP)结构模板

> 篇幅参考:**20-40 页**;核心是**逻辑闭环**:痛点 → 方案 → 市场 → 模式 → 竞争 → 团队 → 财务 → 落地 → 风险。

| # | 模块 | 写作要点 | 常见雷区 |
|---|---|---|---|
| 1 | **执行摘要** | 全文浓缩,1-2 页;最后写但放最前 | 写成目录流水账 |
| 2 | **项目背景与痛点** | 用数据+故事讲清「谁、什么痛、有多痛」 | 只讲现象不讲规模 |
| 3 | **产品/服务方案** | 我们提供什么、怎么解决痛点;配原型图/demo | 只讲功能不讲价值 |
| 4 | **技术/创新点** | 核心壁垒:专利、算法、独家资源、模式创新 | 堆术语无实质 |
| 5 | **市场分析** | 市场规模(TAM/SAM/SOM)、增长趋势、目标用户画像 | 数据无出处 |
| 6 | **竞争分析** | 直接/间接竞品对比;我们的差异化定位 | 说「我们没有竞争对手」 |
| 7 | **商业模式** | 怎么赚钱:收入来源、定价、成本结构、单元经济 | 只说「卖产品」不拆解 |
| 8 | **营销与运营** | 获客渠道、运营策略、推广计划 | 空喊「全网营销」 |
| 9 | **团队介绍** | 成员互补、与项目匹配、导师背书 | 罗列简历不讲匹配 |
| 10 | **财务预测** | 3 年收入/成本/利润预测、盈亏平衡、资金需求 | 数字凭空捏造无逻辑 |
| 11 | **落地与进展** | 已做的事:试点、订单、合作、资质 | 一张白纸无验证 |
| 12 | **社会价值** | 对接五大发展理念、就业/公益/带动效应 | 硬贴标签无实质 |
| 13 | **风险与对策** | 技术/市场/政策/财务风险及应对 | 只列风险不给对策 |

> 评分项里「执行摘要」「商业模式」「团队」「财务」是权重高地,优先打磨这四块。

## 6. 团队分工(RACI 简版)

> R=负责执行 A=最终拍板 C=需咨询 I=知晓

| 任务 | 项目负责人 | 技术/产品 | 市场/运营 | 财务 | 设计/文案 | 指导老师 |
|---|---|---|---|---|---|---|
| 选题决策 | A/R | C | C | I | I | C |
| 用户调研 | A | I | R | I | C | C |
| 产品原型 | A | R | C | I | C | C |
| 商业模式 | A | C | R | C | I | C |
| 财务模型 | A | I | C | R | I | C |
| BP 撰写 | A | C | C | C | R | C |
| 路演 PPT | A/R | C | C | I | R | I |
| 答辩主讲 | R | I | I | I | I | C |

> 经验:项目负责人(CEO)必须深度参与每一个模块,**不能只当「挂名队长」**——评委一眼能看出谁是真的在掌舵。

## 附录 A · 往届金奖项目的共性特征

1. **选题切口小而痛**:解决一个具体人群的具体问题,而非宏大叙事。
2. **有真实落地证据**:试点数据、订单、合作协议、营收,哪怕规模小。
3. **团队高度互补且匹配**:成员背景与项目强相关,不是临时凑的。
4. **数据全部可溯源**:每个关键数字都有出处。
5. **故事讲得好**:路演能让人 30 秒内 get 到「这事值、这事能成、就该他们做」。
6. **对接政策/社会价值自然**:不是硬贴标签,而是项目本身就长在政策方向上。

## 附录 B · 常见答辩「杀手锏」问题(提前准备答案)

- 你们和 XX(知名竞品)比,凭什么赢?
- 这个市场这么大,为什么大公司不做?
- 你们的财务数据是怎么测算的?依据是什么?
- 团队里谁是真正懂技术的?核心壁垒在哪?
- 如果融不到钱/拿不到奖,你们项目还能活下去吗?
- 你们说的「社会价值」,具体体现在哪几个数字上?
- 这个项目毕业了你们还做吗?(考察真实承诺)

## 附录 C · 术语速查

- **大挑 / 小挑**:课外学术科技作品竞赛 / 创业计划竞赛,交替举办。
- **BP**:Business Plan,商业计划书。
- **MVP**:Minimum Viable Product,最小可行产品。
- **TAM/SAM/SOM**:总潜在市场 / 可服务市场 / 可获取市场。
- **路演**:限时演讲+答辩,小挑核心展示环节。
- **五大发展理念**:创新、协调、绿色、开放、共享——对应五大组别。
